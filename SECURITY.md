# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Pori, please report it responsibly.

**Do not open a public issue.**

Instead, email **sathekgealoy@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

This policy applies to:

- The `pori` Python package
- The Pori Cloud API (`pori_cloud`)
- Official tools and integrations

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
| < 1.0   | No        |
