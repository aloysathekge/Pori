# Progress Tracking

Generated by Aloy Agent Kit. Safe to regenerate.

Use the shared Aloy progress rule:

- .agent/progress/current.md
- .agent/progress/handoffs/

- ../Aloy_Agent_kit/agent-kit/skills/progress-tracking/SKILL.md

Rule:

- Before meaningful work, read .agent/progress/current.md.
- After meaningful work, update .agent/progress/current.md if the next session should know what changed.
- If work is complete or paused after a large change, create a handoff in .agent/progress/handoffs/.
- Update AGENTS.md only when stable onboarding information changes.
- Do not finish meaningful work without checking whether progress, handoff, or AGENTS.md needs an update.
