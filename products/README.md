# products/ — Agent products built on Pori

Each product = **kernel (`pori`) + chosen extensions (`ext/pori-*`) + a niche composition** (persona, tool selection, validators/policies, memory config, surfaces). Products may import `ext` and `pori`; nothing imports products.

- **`aloy/`** — FLAGSHIP, product #1: a personal + org OS agent (Hermes-class capabilities and more).

Future products are thin niche compositions on the same kernel (e.g. a research agent, a coding agent, a support agent) — not new frameworks. New products are added here.

See [`../MONOREPO.md`](../MONOREPO.md) and [`../docs/Pori.md`](../docs/Pori.md) §8.
